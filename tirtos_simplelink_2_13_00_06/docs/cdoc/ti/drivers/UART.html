typedef enum UART_LibType {
    UART_LibType_Instrumented,
    // instrumented
    UART_LibType_NonInstrumented
    // non-instrumented
} UART_LibType;
 
VALUES
LibType_Instrumented — The library supplied is prebuilt with logging and assertions enabled.
LibType_NonInstrumented — The library supplied is prebuilt with logging and assertions disabled.
DETAILS
This enumeration defines all the UART library type provided by the product. You can select the library type by setting the UART.libType configuration parameter.

metaonly config UART.libType  // module-wide
index URL

UART Library type

Configuration settings
UART.libType = UART.LibType UART.LibType_Instrumented;
 
DETAILS
Deprecated: Please use ti.tirtos.TIRTOS.libType instead.
The UART runtime is provided in the form of a library that is linked with your application. Several forms of this library are provided with the UART product. This configuration parameter allows you to select the form of UART to use.
The default value of libType is UART_LibType_Instrumented. For a complete list of options and what they offer see LibType.
