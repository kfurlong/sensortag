typedef enum SDSPI_LibType {

    // instrumented
    SDSPI_LibType_NonInstrumented
    // non-instrumented
} SDSPI_LibType;
 
VALUES
LibType_Instrumented — The library supplied is prebuilt with logging and assertions enabled. Diags_USER1 mask logs basic information Diags_USER2 mask logs detailed information
LibType_NonInstrumented — The library supplied is prebuilt with logging and assertions disabled.
DETAILS
This enumeration defines all the SDSPI library type provided by the product. You can select the library type by setting the SDSPI.libType configuration parameter.

metaonly config SDSPI.libType  // module-wide
index URL

SDSPI Library type

Configuration settings
SDSPI.libType = SDSPI.LibType SDSPI.LibType_Instrumented;
 
DETAILS
Deprecated: Please use ti.tirtos.TIRTOS.libType instead.
The SDSPI runtime is provided in the form of a library that is linked with your application. Several forms of this library are provided with the SDSPI product. This configuration parameter allows you to select the form of SDSPI to use.
The default value of libType is SDSPI_LibType_Instrumented. For a complete list of options and what they offer see LibType.
