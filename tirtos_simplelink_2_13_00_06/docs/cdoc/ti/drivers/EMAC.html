EMAC library selection options

C synopsis target-domain
typedef enum EMAC_LibType {
    EMAC_LibType_Instrumented,
    // instrumented
    EMAC_LibType_NonInstrumented
    // non-instrumented
} EMAC_LibType;
 
VALUES
LibType_Instrumented — The library supplied is prebuilt with logging and assertions enabled.
LibType_NonInstrumented — The library supplied is prebuilt with logging and assertions disabled.
DETAILS
This enumeration defines all the EMAC library type provided by the product. You can select the library type by setting the EMAC.libType configuration parameter.

Configuration settings
EMAC.libType = EMAC.LibType EMAC.LibType_Instrumented;
 
DETAILS
Deprecated: Please use ti.tirtos.TIRTOS.libType instead.
The EMAC runtime is provided in the form of a library that is linked with your application. Several forms of this library are provided with the EMAC product. This configuration parameter allows you to select the form of EMAC to use.
The default value of libType is EMAC_LibType_Instrumented. For a complete list of options and what they offer see LibType.
