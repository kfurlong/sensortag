typedef enum I2C_LibType {
    I2C_LibType_Instrumented,
    // instrumented
    I2C_LibType_NonInstrumented
    // non-instrumented
} I2C_LibType;
 
VALUES
LibType_Instrumented — The library supplied is prebuilt with logging and assertions enabled. Diags_USER1 mask logs basic information Diags_USER2 mask logs more detailed information
LibType_NonInstrumented — The library supplied is prebuilt with logging and assertions disabled.
DETAILS
This enumeration defines all the I2C library type provided by the product. You can select the library type by setting the I2C.libType configuration parameter.

DETAILS
Deprecated: Please use ti.tirtos.TIRTOS.libType instead.
The I2C runtime is provided in the form of a library that is linked with your application. Several forms of this library are provided with the I2C product. This configuration parameter allows you to select the form of I2C to use.
The default value of libType is I2C_LibType_Instrumented. For a complete list of options and what they offer see LibType.
